2) Kontaktformular – Empfänger & Fallback
	•	Primärziel: Formular nutzt mailto: an leersch.pl.imo.
	•	Passe den Submit‑Handler an:

const form=document.querySelector('form[action^="mailto:"], form#contactForm');
if(form){
  form.addEventListener('submit',function(e){
    const name=document.getElementById('name')?.value||'';
    const email=document.getElementById('email')?.value||'';
    const msg=document.getElementById('message')?.value||'';
    const body=`Name: ${name}\nE-Mail: ${email}\n\nNachricht:\n${msg}`;
    this.action=`mailto:leersch.pl.imo?subject=${encodeURIComponent('Kontaktanfrage von '+name)}&body=${encodeURIComponent(body)}`;
  });
}


	•	Fallback: Falls mailto: mit dieser Adresse nicht öffnet (Heuristik: setTimeout 800 ms nach submit, wenn Seite/Client nicht wechselte), blende das Formular aus und zeige stattdessen drei Buttons an derselben Stelle:
	•	„E‑Mail schreiben“ → mailto:lersch@plimmo.de
	•	„WhatsApp“ → https://wa.me/4915201393868
	•	„Rückruf anfordern“ → tel:015201393868
(Buttons im bestehenden Stil .btn, .btn-whatsapp nutzen.)
